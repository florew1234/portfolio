Promeview(My Rotten Tomatoes)

SUMMARY

Promeview (My rotten tomatoes) is a web application that provides the user with information on films available on tmdb or not. When a user is not logged in, he can only see the films available on the platform, but once logged in, he can see the description of a film, give his opinion, rate it or see the opinions and ratings of other cinephiles

TECH

  • Nuxt.js
  • MongoDB

Commet (YOWL)

SUMMARY

Commet means: Comment Everything. It's a platform where authenticated users can express themselves freely by leaving a comment on any content available on the Internet. On Commet, users can leave a comment, edit or delete it, and view other users' comments. But if they are not authenticated, they can only read the comments.

TECH

  • Laravel Restful API
  • Vue.js
  • Mysql
Visit Commet on Github

Why This Project?

This project was designed to immerse us in the development of a full stack application by combining two powerful technologies: Laravel to create a RESTful API and Vue.js for the front-end. The main objective was to familiarize ourselves with the creation and use of an API, which is essential in modern web application development. By bringing these two technologies together, we were able to explore best practice in communication between back-end and front-end, while developing skills in the concepts of REST and microservices architecture. Teamwork was a fundamental aspect of this project.

What I've Learned

  • Project planning and ideation using user stories and wireframe design using Figma.
  • Project management with Agile methodologies,
  • Setup background jobs using mysql server for database syncing with user's account.
  • Applied RESTful principes for backend architecture.
  • Mobile-first design for front-end with Tailwind CSS capabilities.
